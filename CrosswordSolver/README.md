# CrosswordSolver

Программа для разгадывания кроссвордов по заданному шаблону.

## Требования

- Компилятор с поддержкой C++20
- CMake версии 3.10 или выше

## Сборка и запуск

```bash
mkdir build
cd build
cmake ..
cmake --build .
./CrosswordSolver
```

## Тестирование

```bash
cd build
ctest
```
